================================================================================
TROUBLESHOOTING
================================================================================

Issue: "Failed to import FDS to FDB converter"
Solution: Ensure fds_to_fdb_converter.py is in fds_workflow/ directory

Issue: "No FDS simulation outputs found"
Solution: Run FDS simulations first, wait for completion

Issue: "Conversion failed"
Solution: Check error message in status display or database

Issue: "Database locked"
Solution: Close other QRA System instances, restart
