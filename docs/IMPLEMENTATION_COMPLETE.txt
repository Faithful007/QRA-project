FEATURES IMPLEMENTED
================================================================================

[X] FDS Version Selection (FDS5/FDS6)
    - Radio buttons in UI
    - FDS6 set as default
    - Backward compatibility with FDS5

[X] Fuel Type Selection
    - 5 fuel types: Petrol, Diesel, CNG, LPG, EVC
    - Dropdown menu in UI
    - Diesel set as default

[X] Editable Fuel Properties
    - SOOT_YIELD (0.0-1.0, 3 decimals)
    - CO_YIELD (0.0-1.0, 3 decimals)
    - HEAT_OF_COMBUSTION (1E7-1E8 J/kg)
    - Editable for Petrol/Diesel only
    - Fixed for CNG/LPG/EVC

[X] Automatic Fuel Selection
    - Monitors HRR checkbox changes
    - Auto-selects Diesel for HRR >= 20MW
    - Auto-selects Petrol for HRR < 20MW
    - Respects manual CNG/LPG/EVC selections

[X] FDS Generator Updates
    - FireScenario dataclass extended
    - generate_reaction() supports FDS5/FDS6
    - Fuel-specific REAC namelists

USAGE INSTRUCTIONS

1. Run the QRA System application:
   $ cd /home/ubuntu/qra_system_v2
   $ python3 qra_main_app.py

2. Navigate to "Generate FDS" tab

3. Select FDS version (FDS6 recommended)

4. Choose HRR values (system auto-selects fuel)

5. Optionally override fuel type

6. Adjust fuel properties if using Petrol/Diesel

7. Generate FDS files
