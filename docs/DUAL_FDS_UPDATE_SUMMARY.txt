UPDATE DATE: February 12, 2026

PROBLEM SOLVED:
---------------
FDS5 and FDS6 installations conflicting on the same system due to:
- PATH environment variable conflicts
- DLL version conflicts
- Registry conflicts

SOLUTION IMPLEMENTED:
---------------------
Batch file execution method that isolates each FDS version's environment.

EXECUTION LOGIC:
----------------
1. User selects FDS version (FDS5/FDS6) in Tab 2
2. System checks for corresponding batch file in Tab 3
3. If batch file exists: Use batch file execution
4. If not: Fall back to direct executable
5. If neither: Show error with guidance

BATCH FILE EXECUTION:
---------------------
Command: cmd /c run_fds6.bat && fds input.fds
Effect: Sets isolated PATH before running FDS
